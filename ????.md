# MQTT WebSocket Bridge - 部署与使用说明

## 1️⃣ 部署到 Cloudflare

```bash
# 安装依赖
npm install

# 登录 Cloudflare
npx wrangler login

# 配置环境变量（可按需修改默认 Broker）
npx wrangler secret put MQTT_BROKER_HOST
# 输入: broker.emqx.io
npx wrangler secret put MQTT_BROKER_PORT
# 输入: 1883
# 如需鉴权
npx wrangler secret put MQTT_AUTH_TOKEN
# 输入: 你的密钥

# 部署
npx wrangler deploy
```

部署成功后输出：
```
Uploaded mqtt-websocket-bridge (5.33 KiB)
Published at https://mqtt-websocket-bridge.你的子域名.workers.dev
```

## 3️⃣ 连接不同 Broker（无需改代码）

部署后通过 URL 参数动态切换目标 Broker：

```js
// 默认使用 wrangler.jsonc 中配置的 Broker
wss://mqtt-websocket-bridge.你的子域名.workers.dev/mqtt

// 切换到 mosquitto 测试服务器
wss://.../mqtt?host=test.mosquitto.org&port=1883

// 切换到 TLS 加密的 EMQX
wss://.../mqtt?host=broker.emqx.io&port=8883&tls=true

// 切换到你自己的本地 Broker（需公网可达）
wss://.../mqtt?host=你的服务器IP&port=1883
```

## 4️⃣ 完整链路：ESP8266 → 浏览器

### 步骤 A: ESP8266 发布数据

```cpp
#include <ESP8266WiFi.h>
#include <PubSubClient.h>

WiFiClient espClient;
PubSubClient client(espClient);

void setup() {
  WiFi.begin("你的WiFi", "密码");
  client.setServer("broker.emqx.io", 1883);
  client.connect("esp8266_" + String(random(0xffff), HEX));
}

void loop() {
  client.loop();
  float temp = 25.5;
  client.publish("sensor/temp", String(temp).c_str());
  delay(5000);
}
```

### 步骤 B: 浏览器通过 Worker 订阅并展示

```html
<!DOCTYPE html>
<html>
<body>
  <h1>温度: <span id="temp">--</span>°C</h1>
  <script src="https://unpkg.com/mqtt/dist/mqtt.min.js"></script>
  <script>
    const client = mqtt.connect(
      'wss://mqtt-websocket-bridge.你的子域名.workers.dev/mqtt'
    )

    client.on('connect', () => {
      console.log('已通过 Worker 连接到 Broker')
      client.subscribe('sensor/temp')
    })

    client.on('message', (topic, payload) => {
      document.getElementById('temp').textContent = payload.toString()
    })
  </script>
</body>
</html>
```

## 5️⃣ 客户端连接方式

### MQTT.js（推荐）

```js
// 浏览器
const client = mqtt.connect('wss://你的worker名.workers.dev/mqtt', {
  clientId: 'web_' + Math.random().toString(16).substr(2, 8),
  clean: true,
})

client.on('connect', () => {
  console.log('已连接')
  client.subscribe('sensor/#')
  client.publish('sensor/temp', '25.5')
})

client.on('message', (topic, payload) => {
  console.log(topic, payload.toString())
})
```

```js
// Node.js
const mqtt = require('mqtt')
const client = mqtt.connect('wss://你的worker名.workers.dev/mqtt')
```

### Paho MQTT（浏览器）

```js
const client = new Paho.MQTT.Client(
  '你的worker名.workers.dev',
  Number(443),
  '/mqtt',
  'clientId'
)

client.connect({
  useSSL: true,
  onSuccess: () => {
    client.subscribe('sensor/#')
    client.send('sensor/temp', '25.5')
  }
})
```

### Mosquitto 命令行

```bash
mosquitto_sub -h "你的worker名.workers.dev" -p 443 --ws-ca /mqtt -t "sensor/#" -v
mosquitto_pub -h "你的worker名.workers.dev" -p 443 --ws-ca /mqtt -t "sensor/temp" -m "25.5"
```

## 6️⃣ 鉴权

```bash
npx wrangler secret put MQTT_AUTH_TOKEN
# 输入: mySecretToken123
```

客户端携带 Token（通过自定义 Header，MQTT.js 自动处理）：

```js
const client = mqtt.connect(
  'wss://mqtt-websocket-bridge.你的子域名.workers.dev/mqtt',
  {
    clientId: 'web_client',
    clean: true,
    username: 'admin',   // MQTT Broker 用户名（如有）
    password: 'public',  // MQTT Broker 密码（如有）
  }
)
```

## 7️⃣ 架构总览

```

│  ESP8266     │────────────────▶│  MQTT Broker  │

                                        │
                           TCP 连接     │
                                        ▼
                               ┌────────────────┐
                               │ Cloudflare      │
                               │ Worker (桥接)   │
                               │ /mqtt endpoint  │
                               └────────┬───────┘
                                        │
                           WebSocket    │
                           (wss://)     │
                                        ▼
┌─────────────┐                  ┌──────────────┐
│  Web 浏览器  │◀───────────────│  MQTT.js      │
│  (订阅者)    │   subscribe     │  (客户端库)    │
└─────────────┘                  └──────────────┘
```
